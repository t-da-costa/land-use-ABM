\section{Modules}
\subsection{Landscape Generation}

The \emph{landscape} is a rectangular grid of $n_{rows} \times n_{cols}$, containing $K$ plots. \\

\noindent A \emph{plot} is defined by its ID \texttt{PlotID}, its coordinates \texttt{(x,y)}, and its quality \texttt{q}. \\
Von Neumann neighborhood (N, S, E, W) is used to determine the adjacent plots. \\
The Euclidean distance ($d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$) is used to determine the distance between two plots. \\

\noindent The quality of each plot is generated from Gaussian noise with mean $0$, smoothed using a Gaussian kernel with length scale \texttt{q\_length\_scale}, then rescaled using \texttt{q\_sigma}. The quality values are then normalized on [0, 1]. \\

\noindent There are $N < K = n_{rows} \times n_{cols}$ \emph{farmers} in the landscape, each owning a number of plots drawn from a lognormal distribution with parameters \texttt{farm\_mu} and \texttt{farm\_sigma}. Each farmer owns at least one plot, and the total number of plots owned by all farmers equals $K$. 

\noindent Farmers are defined by their ID \texttt{FarmerID} and the list of plots \texttt{plotID} they own: the number of plots owned corresponds to the farm's size $m$.
