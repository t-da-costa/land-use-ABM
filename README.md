# Agent-Based Modeling of Agricultural Land Use and Environmental Policy

This repository now contains a runnable baseline version of the EEB324 project model. The implementation follows the current proposal: a spatial landscape with heterogeneous farmers, three land uses (`I`, `O`, `S`), three decision regimes, environmental feedbacks, and three subsidy designs.

The default presentation package is intentionally narrower than the full model. The main story focuses on `productivist`, `market`, and `policy_practice`. Conversion and results-based subsidies are still implemented, but they are exported as robustness material instead of headline evidence.

## What the model does

- Builds a spatially autocorrelated landscape with clustered farm ownership.
- Simulates farmer decisions under `productivist`, `market`, and `policy`.
- Supports three policy designs inside the policy regime: practice-based, conversion-to-organic, and results-based.
- Updates environmental quality and land quality over time.
- Exports a core presentation package and a separate robustness package.

## Run instructions

Create a local environment and install dependencies:

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
```

Run the default experiment:

```bash
.venv/bin/python ABM-LU-program/abm_model.py
```

Useful options:

```bash
.venv/bin/python ABM-LU-program/abm_model.py --replicates 5 --steps 25
.venv/bin/python ABM-LU-program/abm_model.py --output-dir results/andrew-ethan
```

## Modeling note

The proposal’s results-based subsidy needed one timing clarification to be operational. In the implementation, that payment is based on next-period ecological performance generated by the current land-use choice, so it changes incentives at the moment the farmer makes the decision.

The main summary tables now distinguish:

- profit before subsidy
- subsidy payments
- profit after subsidy

That separation matters because subsidy-inclusive profit is useful for farmer incentives, but it is too flattering to use as the only economic headline.
